# Rock vs Mine Prediction

## Overview
This project uses **Logistic Regression** to classify sonar signals as either **Rock** or **Mine**.

---

## Dataset
- Samples: 208
- Features: 60 numerical features
- Labels:
  - R → Rock
  - M → Mine

---

## ML Workflow
1. Data loading
2. Data preprocessing
3. Train-test split
4. Model training
5. Model evaluation
6. Prediction system
